API detail

Register

Register a new user and log in.

  • Curl
  • HTTP
curl -X POST -d '{
  "user": {
    "username": "test61",
    "email": "test61@test.cz",
    "password": "123456",
    "password_confirmation": "123456",
    "notify_through_email": 0
  }
}
' "http://127.0.0.1:3000/api/v1/register"
POST /api/v1/register HTTP/1.1
Host: 127.0.0.1:3000

{
  "user": {
    "username": "test61",
    "email": "test61@test.cz",
    "password": "123456",
    "password_confirmation": "123456",
    "notify_through_email": 0
  }
}

Login

Log the user in using credentials. Login can be either the username or email address.

  • Curl
  • HTTP
curl -X POST -d '{
    "user": {
        "login": "test59",
        "password": "123456"
    }
}' "http://127.0.0.1:3000/api/v1/login"
POST /api/v1/login HTTP/1.1
Host: 127.0.0.1:3000

{
    "user": {
        "login": "test59",
        "password": "123456"
    }
}

Logout

Log the user out.

  • Curl
  • HTTP

Host: 127.0.0.1:3000
