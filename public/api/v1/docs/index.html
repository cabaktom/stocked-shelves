API detail

Register

Register a new user and log in.

  • Curl
  • HTTP
curl -X POST -d '{
  "user": {
    "username": "test61",
    "email": "test61@test.cz",
    "password": "123456",
    "password_confirmation": "123456",
    "notify_through_email": 0
  }
}
' "http://127.0.0.1:3000/api/v1/register"
POST /api/v1/register HTTP/1.1
Host: 127.0.0.1:3000

{
  "user": {
    "username": "test61",
    "email": "test61@test.cz",
    "password": "123456",
    "password_confirmation": "123456",
    "notify_through_email": 0
  }
}

Login

Log the user in using credentials. Login can be either the username or email address.

  • Curl
  • HTTP
curl -X POST -d '{
    "login": "test30@test.cz",
    "password": "123456"
}' "http://127.0.0.1:3000/api/v1/login"
POST /api/v1/login HTTP/1.1
Host: 127.0.0.1:3000

{
    "login": "test30@test.cz",
    "password": "123456"
}

Logout

Log the user out.

  • Curl
  • HTTP
curl -X DELETE "http://127.0.0.1:3000/api/v1/logout"
DELETE /api/v1/logout HTTP/1.1
Host: 127.0.0.1:3000

Current user

Get the currently logged in user.

  • Curl
  • HTTP
curl -X GET "http://127.0.0.1:3000/api/v1/current_user"
GET /api/v1/current_user HTTP/1.1
Host: 127.0.0.1:3000

Products

All

Get all products of the currently logged in user.

  • Curl
  • HTTP
curl -X GET "http://127.0.0.1:3000/api/v1/products"
GET /api/v1/products HTTP/1.1
Host: 127.0.0.1:3000

Product

Get a product of the currently logged in user.

  • Curl
  • HTTP
curl -X GET "http://127.0.0.1:3000/api/v1/products/8"
GET /api/v1/products/8 HTTP/1.1
Host: 127.0.0.1:3000

Create product

Create a new product.

  • Curl
  • HTTP
curl -X POST -d '{
  "product": {
    "name": "API product",
    "barcode": "123456773"
  }
}
' "http://127.0.0.1:3000/api/v1/products"
POST /api/v1/products HTTP/1.1
Host: 127.0.0.1:3000

{
  "product": {
    "name": "API product",
    "barcode": "123456773"
  }
}

Update product

Update an existing product.

  • Curl
  • HTTP
curl -X PUT -d '{
  "product": {
    "name": "updated API product",
    "barcode": "123456789"
  }
}
' "http://127.0.0.1:3000/api/v1/products/8"
PUT /api/v1/products/8 HTTP/1.1
Host: 127.0.0.1:3000

{
  "product": {
    "name": "updated API product",
    "barcode": "123456789"
  }
}

Delete product

Delete the logged in user’s product.

  • Curl
  • HTTP
curl -X DELETE "http://127.0.0.1:3000/api/v1/products/21/image"
DELETE /api/v1/products/21/image HTTP/1.1
Host: 127.0.0.1:3000
